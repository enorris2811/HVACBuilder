CCG Air Handler Configurator

Design and configuration made easy by following the steps below!

  • Step 1: Login using your email or create a New User Account.
  • Step 2: Create A New Project using the New Project Wizard or select an existing project from the sidebar tree.
  • Step 3: Create A New Unit using the New Unit Wizard.
  • Step 4: Select a unit from the sidebar tree to configure the settings and options
  • Step 5: Once configured, the units points and sequence will be automatically generated and saved.
  • Step 6: Add as many units to the project as needed.

Configured Units can be reconfigured by clicking on the configure link and going through the same process again.
